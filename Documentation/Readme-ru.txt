краткое описание решения DirectFileTransfer

Стандартные папки решения: 

1) .nuget - папка развертывания nuget пакетов
2) Documentation - папка документации 
3) ExternalLibraries - папка внешних библиотек
4) InstallationPackage - папка инсталяционных пакетов нужных для нормальной работы решения
5) packages - папка nuget пакетов
6) Source - исходные тексты проектов решения
7) Tests - тестовые проекты

Содержимое папки Documentation
1) cridentials.txt - параметры доступа к используемым внешним сервисам
2) Readme-ru.txt - краткое описание решения DirectFileTransfer на русском языке
3) Readme-en.txt - краткое описание решения DirectFileTransfer на английском языке

Содержимое папки Source
1) CommonClasses - общие классы используемые в работе
2) Tests - заготовки тестовых классов NUnit
3) ServerSTUNLibrary - основная библиотека решающая поставленную задачу

Содержимое папки Tests
1) Booth.Console - консольный тестовый стенд

алгоритм запуска решения 
1) проверить доступность внешнего ip на роутере
2) проверить доступность UPnP на роутере
3) проверить разрешение UPnP на роутере
4) проверить доступность сети на рабочем ПК
5) собрать решение
6) запустить Booth.Console на выполнение 
7) дождаться надписи "WebAppHost Startup...."
8) запустить браузер
9) зайти в браузере на адрес http://noipdirectfiletransfer.ddns.net:9000/
