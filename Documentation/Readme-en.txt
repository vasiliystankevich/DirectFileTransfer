Solution Brief DirectFileTransfer

Standard folders solutions:

1) .nuget - folder deployment nuget packages
2) Documentation - Documentation folder
3) ExternalLibraries - folder external libraries
4) InstallationPackage - folder of installation packages necessary for the normal operation of the decision
5) packages - packages folder nuget
6) Source - the original text of the draft decision
7) Tests - test projects

The contents of the folder Documentation
1) cridentials.txt - access parameters used external services
2) Readme-ru.txt - brief description of the solutions DirectFileTransfer in Russian
3) Readme-en.txt - brief description of the solutions DirectFileTransfer English

Content Source folder
1) CommonClasses - common classes used in the work
2) Tests - blank NUnit test classes
3) ServerSTUNLibrary - the main library to solve tasks

Content folder Tests
1) Booth.Console - console testbed

the start sequence solutions
1) Check the availability of external ip on the router
2) check the availability of UPnP on the router
3) check the resolution UPnP on the router
4) Check the availability of the network on your PC
5) to collect the solution
6) start to perform Booth.Console
7) wait for the inscription "WebAppHost Startup ...."
8) to launch the browser
9) to enter in the browser address http://noipdirectfiletransfer.ddns.net:9000/
